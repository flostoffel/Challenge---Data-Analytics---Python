Instrucciones al usuario:

## Crear el entorno virtual

    1- instalar la libreria venv con el comando:

    pip install virtualenv


    2- Dirigirse al directorio donde se desea instalar el entorno virtual y ejecutar:

    virtualenv nombre_del_entorno_virtual


    3- Activar el entorno virtual:

    .\nombre_del_entorno_virtual\Scripts\activate


## Clonar este repositorio:

    git clone COLOCAR NOMBRE REPO

### Instalar las librerias necesarias:

    librerias.txt

## Configurar las URL de descarga: 
    
    Modificar en caso de ser necesario en src/url.py
    
## Configurar la conexión con la base de datos: 

    abrir el archivo env.py y completar los datos de conexión

## Ejecutar:
    
    python main.py
